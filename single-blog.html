I worked on a piece of software at my university called Lost Item Management. The software confirmed that students had published images of their lost items and that the owners of the lost items had found and claimed it .

This project designs and implements a web-based distributed lost and found service.The main goal of this project is to enable students on campus to use this web service to find lost items more quickly. This application is an online discovery portal with an interactive user interface. An easy-to-use web application built using the database-connected web programming language. This project is designed according to the client-server model.

The main features of the Web application include an authentication, registration, and login system, and a Lost and Found page where users can enter information about lost items. Due to some security concerns, users cannot delete items. The reason for is that some users may delete items simply because they are playing around. This privilege is only granted to the administrator

The application also allows the user to update the listed items as needed. User can update contact information and change passwords. This project uses HTML, CSS and JavaScript for the client side interface. PHP is used to connect the front-end web pages to the MySQL database server. MySQL server as a database for maintaining and storing data and information
